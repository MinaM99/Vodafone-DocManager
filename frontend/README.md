# Vodafone-DocManager Frontend

## Setup & Deployment

### 1. Install Dependencies

```bash
npm install
```

### 2. Start the Development Server

```bash
npm start
```

### 3. Build for Production

```bash
npm run build
```

### 4. Deploy to Tomcat

```bash
cp -r build/* <TOMCAT_HOME>/webapps/Vodafone-DocManager
```
